## 更新履歴

:::changelog

### v2.0.0

- ギミックの一部ロジックをリファクタリング。
- 石碑モデルの位置とサイズをプレビューするための Editor Only インジケーターメッシュを追加。
- モデル本来の AFK と RIP AFK を切り替えるための任意スイッチ機能を追加。
- カスタムパラメーター用の簡易スイッチ作成機能を追加し、手動でスイッチを作成せずに簡単に RIP AFK を切り替えられるように改善。
- アニメーション内で指定コンポーネントを含まないオブジェクトを強制的に無効化する `強制OFF項目` リストを追加。主に特定の Write Defaults Off 問題へ手動対応するための機能。
- 特定の状況で石碑モデルが常に表示されたままになる問題を修正。
- 他プレイヤーの視界外で AFK に入る／解除した際、Animator のカリングによってプレイヤーが透明化する問題を修正。
- Write Defaults Off を使用している Avatar が、他プレイヤーの視界外で RIP AFK を解除すると透明化する問題を修正。

:::

:::changelog

### v1.1.1

- AFK 状態中に新しいプレイヤーがインスタンスに参加した際、アバターが見えなくなる問題の防止を試みました。

:::

:::changelog

### v1.1.0

- カスタムパラメーター機能を追加し、メニューのトグルやその他のロジックからAFKアニメーションを再生できるようにしました。

:::
